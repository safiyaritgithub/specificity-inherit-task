CSS SELECTORS SPECIFICITY

.container .box: specificity = (0, 2, 0) → 2 class selectors

#main-content .box: specificity = (1, 1, 0) → 1 ID selector, 1 class selector

div.box: specificity = (0, 1, 1) → 1 class selector, 1 element selector

Inheritance in CSS:

CSS properties can be inherited by child elements from their parent elements. This means that if a parent element has a certain style applied to it, its child elements will inherit that style unless explicitly overridden.
